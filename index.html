MetaCyc Consists of a database of non-redundant and experimentally elucidated metabolic pathways and enzymes. Pathways, enzymes, reactions, and metabolites. Predict sequenced gene metabolic pathways, support enzyme database metabolic engineering, facilitates studies into metabolomics. Contains 2859 pathways from 3185 different organisms. https://metacyc.org/MetaCycUserGuide.shtml Website and browser search bar for instant searching, or by downloading MetaCyc data files https://metacyc.org/
BIGG Model Combines a collection of stardized identifiers (BiGG IDs) into a single pool of over 70 publicated metabolic genome-scale networks. Metabolites, reaction, gene, or organism. Map the genes to the NCBI annotations, and various external databases are linked to metabolites (KEGG, PubChem, and many more). Improve the quality of annotations and standardization in the model, without modifying the content of the reaction. The current knowledge base includes 108 GEMs metabolic knowledge. https://github.com/SBRG/bigg_models/releases Website or web API http://bigg.ucsd.edu/
KEGG Enables users to understand the high-level biological system functions and resources based on the molecular-basis knowledge (genome sequencing datasets). Pathway, KO functional orthologs, genes and proteins, reactions and enzyme nomenclature. Connecting genomes to metabolic pathways using the Enzyme Commission (EC) number. Chemical information - Contains 548 (851, 821) pathways, 11,632 reactions, and 7,890 enzymes, from 680 eukaryotes, 6623 bacteria, 363 archaea (updated: June 2021). https://www.genome.jp/kegg/kegg1a.html API and KEGG FTP site (academic users) https://www.kegg.jp/kegg/
ModelSEED Consists of a biochemical database (incorporated and unified from many major external sources including MetaCyc, KEGG and BiGG) designed for standardisation and data integration assistance. Compound and reaction table through GUI. Compares and incorporates annotations from multiple resources and databases. Describes redundancies and thermodynamics computing properties. ModelSEED now includes 33,978 compounds and 36,645 reactions. https://github.com/ModelSEED/ModelSEEDDatabase Web interfaces (manual searching) or programmatic REST APIs (public access to the Apache Solr database) https://modelseed.org
MEMOSys 2.0 Represents a polyvalent database able to maintain, store and build GEMs. Model ID, organisms, and taxonomy. Builds model hierarchies whose distribution changes automatically. Apply additional annotations, which can be derived directly from the (SBML) format, to all stored components. The updated database now includes 20 publicly accessible versions, which can be quickly exported into structured formats for further study. https://memosys.i-med.ac.at/MEMOSys/help.seam Website and ready-to-use virtual machine image https://memosys.i-med.ac.at/
TCDB Consists of a curated database containing sequences, classification, structural, functional and evolutionary information on transport systems of diverse live species. By sequence or TC number and full-text searching. Categorizes membrane transport proteins, using information on the TC system. The TC method is similar to the EC approach for enzyme categorization, because of how it includes both functional and phylogeny information. The database contains 20892 protein sequences classified into 1565 transporter families based on the TC system. https://www.tcdb.org/browse.php Website https://www.tcdb.org/
