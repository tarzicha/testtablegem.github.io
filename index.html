BIGG Model Combines a collection of starred identifiers (BiGG IDs) into a single pool of over 70 publicated metabolic genome-scale networks. Browse the content of genome-scale models (metabolites, reaction, gene, or organism). Map the genes to the NCBI annotations, and various external databases are linked to metabolites (KEGG, PubChem, and many more). Improve the quality of annotations and standardization in the model, without modifying the content of the reaction. The current knowledge base includes 108 GEMs metabolic knowledge https://github.com/SBRG/bigg_models/releases BiGG can be browsed on the URL website or accessed using a web API http://bigg.ucsd.edu/

TCDB A Web open, curated and relation-based database containing sequence, classification, structural, functional and evolutionary information on transport systems of diverse live species is the Transporter Classification Database (TCDB). Search by sequence or by TC number and full-text searching The database TCDB contains information on the Transporter Classification (TC) system, which is a categorization method for membrane transport proteins. The TC method is similar to the Enzyme Commission (EC) approach for enzyme categorization, because of how it includes both functional and phylogeny information. The database contains 20892 protein sequences These proteins are classified into 1565 transporter families based on the TC-system https://www.tcdb.org/browse.php Database of the transport classification TCDB is the Online web resource (http://www.tcdb.org) that is freely available. https://www.tcdb.org/
